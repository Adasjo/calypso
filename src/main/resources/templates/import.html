Facebook-import

Eftersom det numera krävs en formell jävla ansökan för att slå upp publika event i Facebooks kassa API är denna funktion disabled tills vidare. Förlåt infom :(((

Viktig information

Du kan endast importera publika event som är synliga för alla Facebookanvändare.

Importen kommer att automatiskt skapa ett utkast med samma titel, beskrivande text och tid som ditt Facebookevent, samt länk till själva eventet. Om den beskrivande texten innehåller fem eller mer förekomster av två tecken (t.ex. == eller -•) kommer detta att användas för att dela upp texten i svenska och engelska.

Eventet publiceras inte förrän du granskat och godkänt det.

Observera att eventplats inte följer med i importen utan måste fyllas i manuellt.
